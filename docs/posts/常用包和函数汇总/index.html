数据摘要和清洗

gtExtras

数据集摘要：gtExtras::gt_plt_summary()，展现每个变量的描述性统计指标，并用可视化的方式展现变量的分布。

janitor

列名清洗: janitor::clean_names()，将命名不一致的列名统一规范化。

数据可视化

hrbrthemes

ggplot 主题风格拓展，使用方法可见 hrbrthemes：最好用的 ggplot 主题扩展包。

gghighlight

帮助高亮图表，使用方法可见使用 gghighlight 制图。

waffle

华夫饼图绘制包，使用方法可见使用 waffle package 制作华夫饼图。

ggalluvial

桑基图和凹凸图绘制包，使用方法可见使用 ggalluvial 制作桑基图和凹凸图。

ggwordcloud

用 ggplot 语法绘制词云，使用方法可见ggwordcloud: a word cloud geom for ggplot2及R Charts: Word cloud in ggplot2 with ggwordcloud。

patchwork

快速组合不同图表：p1 + p2。使用方法详见教程。
